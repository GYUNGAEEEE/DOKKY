## 논의 내용

    - 강사님과 논의를 통해 게시판 메뉴를 간소화함
    - 벤치마킹 모델 선정 → OKKY
    - 각 페이지에 필요한 기능을 필수기능과 부가기능으로 나누었음
    - 각자 개발할 페이지를 정함
    
    * 공통 필수기능: 글 작성, 조회, 수정, 삭제 → CRUD
    1. Q&A 게시판(장성욱) 
      - 필수기능: 검색, 댓글
      - 부가기능: 키워드(태그) 검색, 조회수, 좋아요, 댓글수, 글 내용 한줄 미리보기
      
    2. 자유 게시판(김민준)
      - 필수기능: 검색, 댓글
      - 부가기능: 조회수, 좋아요, 댓글수
      
    3. 점심 메뉴 추천 게시판(오주리)
      - 필수기능: 갤러리형 글 목록
      - 부가기능: 좋아요, 지도 api
      
    4. 스터디 모집 게시판(이윤서)
      - 필수기능: 검색, 댓글
      - 부가기능: 조회수, 좋아요, 댓글수
      
    6. 로그인(김경애)
      - 필수기능: 아이디/비밀번호 찾기, 회원가입 버튼
      - 부가기능: 자동 로그인
      
    7. 회원가입(최원준)
      - 필수기능: 아이디 중복 체크, 비밀번호 재확인
      - 부가기능: SNS 간편 회원가입, 탈퇴
      
    8. 마이페이지
      * 일반회원(최원준)
      - 필수기능: 정보 수정 페이지
      - 부가기능: 내가 쓴 글 목록
      * 관리자(김경애)
      - 필수기능: 회원 관리, 글 삭제
      - 부가기능: 회원 승인
